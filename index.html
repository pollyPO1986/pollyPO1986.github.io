About Me

李清瀠Polly Lee

網頁前端工程師

polly0215yu@gmail.com

我叫李清瀠，喜歡學習新事物，對於自己有興趣的領域，都會自行收集資料及自學，工作後的空檔都還會去進修網頁前端相關的課程。

在前端設計的工作經驗約 5 年，前端工程的經驗約 1.5 年左右。

主要的工作範圍有：負責網頁或系統介面的前置討論及部分 UX 動線規劃、介面 UI 切板及整體網頁的動態效果、串接 API 綁定畫面資料等。 主要開發是使用 Vue 框架，近期團隊導入 Electron 框架開發桌面應用軟體，整合公司工廠系統、品牌客戶檢測系統。 團隊也導入 Scrum 敏捷開發流程，軟體開發工作都是環環相扣，敏捷開發讓團隊工作分配與因應客戶需求快速變化上有顯著效果。

本身對於網頁的視覺 UI、使用者互動的 UX 有熱情，對於創作有質感的畫面與操作動線順暢的產品有追求，未來希望可以更往這個方向學習與前進。
